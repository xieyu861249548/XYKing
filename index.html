企业级知识库智能 Agent

B端 SaaS / RAG架构 / 0-1孵化

LangChain 向量检索 多模态解析

背景与挑战：

企业内部文档分散、格式繁杂，传统检索效率低下，客服与销售培训成本极高，知识流转断层。

解决方案与我的角色：

作为产品一号位，主导设计基于 RAG（检索增强生成）的企业知识问答方案。设计了从“文档解析清洗-向量化存储-语义召回-大模型生成”的全链路产品机制。引入了引用溯源（Source Tracking）设计以解决 AI 幻觉问题，增强企业信任感。

商业成果：

  • 上线 3 个月完成 50+ 家中大型企业客户 私有化部署与 SaaS 交付。
  • 将企业内部知识检索时间从平均 10 分钟缩短至 10 秒内。
  • 使得客户企业新员工培训周期缩短 30%，知识复用率大幅提升。

AIGC 营销素材生成平台

商业化变现 / Stable Diffusion / 工作流

Prompt Engineering A/B Test 订阅制变现

背景与挑战：

中小电商卖家在商品图、营销海报制作上成本高昂，且难以掌握复杂的底层绘图 AI 工具，需要一款开箱即用的傻瓜式 AIGC 产品。

解决方案与我的角色：

作为核心产品经理，将底层复杂的生图模型能力封装为场景化的“一键生成”模板。通过深度调优 Prompt 和 ControlNet 工作流，保障生成图像的商品特征一致性。设计了按量付费（Token）与高级会员订阅混合的商业模型。

商业成果：

  • 发布首月实现盈亏平衡，三个月内月度经常性收入(MRR)突破 150万人民币。
  • 累积服务 10万+ 电商卖家，日均生成素材数超 50万张。
  • 相较传统外包设计，为单一商户平均降低 80% 视觉物料成本。

多模态智能客服大模型升级

降本增效 / NLP / 意图识别

策略产品 指标体系 大模型微调

背景与挑战：

原基于决策树的智能客服解决率遇颈（<30%），转人工率居高不下，人力成本承压，且无法处理复杂的长文本语境和情绪安抚。

解决方案与我的角色：

推动原有客服系统向大模型架构重构。构建了数十万级高质量客服语料库协同算法微调。设计“大模型兜底+业务 API 调用（Agent化）”双轨机制，让 AI 能够不仅回答问题，还能直接执行查物流、退款等操作指令。

商业成果：

  • 核心业务线智能解决率从 28% 跃升至 65%，转人工率下降 40%。
  • 每年为集团节省人工座席成本超 千万级。
  • 用户满意度评分（CSAT）提升 15%，模型情绪识别与安抚话术获得用户好评。
